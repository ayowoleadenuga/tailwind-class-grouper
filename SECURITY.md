# Security Policy

## Supported Versions
- Latest published versions of `eslint-plugin-tailwind-group` and `prettier-plugin-tailwind-group`.

## Reporting a Vulnerability
- Email: ayowoleadenuga@gmail.com (or raise an issue here on Github).
- Please include repro steps, expected vs actual behavior, and environment details.
- Do not open public issues for suspected vulnerabilities.

## Response
- We will acknowledge receipt within 3-5 business days and provide a timeline for a fix or mitigation.
